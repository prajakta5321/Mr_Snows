# 🍦 Mr. Snow's Ice Cream Shop

A modern, responsive web application for an ice cream parlor. This project demonstrates a full CRUD (Create, Read, Update, Delete) workflow using a NoSQL database.

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3, and **Bootstrap 5** for a mobile-friendly design.
* **Backend:** **Python 3.x** & **Django Framework**.
* * **frontend database :** **Python 3.x** & .
* **Database:** **MongoDB** (using `djongo` or `pymongo` for integration).
* **Environment:** Virtual Environments (`venv`) for dependency management.

## ✨ Key Features

* **Flavor Gallery:** Browse a wide variety of ice cream flavors with real-time updates from the database.
* **Order Management:** Users can select flavors and quantities (with tracking).
* **Admin Dashboard:** Leveraging Django's built-in admin to manage the ice cream inventory easily.
* **Contact System:** A functional contact form for customer inquiries.
* **Cart sytem product:**Users can select flavors and quantities.
  

## 📂 Project Structure

* `Mr_Snows/` - The main Django project settings.
* `IceCreamApp/` - The core app containing logic for flavors and views.
* `templates/` - HTML files for the website layout.
* `static/` - CSS, Images, and JavaScript files.
